Lightweight macOS Editor

Code

A small, focused editor for the moments when you just need to open a file, make a change, and move on.

If you ever need a basic code editor on a VM, a fresh machine, or anywhere a full IDE feels excessive, this is built for that job. It stays intentionally small and keeps the experience centered on editing files quickly.

Why It Exists

Sometimes you do not need workspaces, extensions, onboarding flows, or a heavyweight development environment. You just need to open a script, inspect a config file, or patch a small issue without turning the task into a setup project.

What It Does Well

  • Open folders and individual files
  • Edit scripts and text files in multiple tabs
  • Remember your window state and recent session
  • Handle basics like find, replace, encoding, and line endings
  • Offer syntax highlighting for common scripting workflows

Built for Quick Tasks

Code is aimed at practical, everyday edits:

  • Updating a shell script on a server or VM
  • Editing a `.env`, Python, or PowerShell file locally
  • Reviewing logs, configs, or small project folders
  • Making a fast change without opening a full IDE

What It Is Not

This is not trying to replace VS Code, Xcode, or any advanced editor.

  • No plugin ecosystem
  • No advanced project tooling
  • No “IDE replacement” ambitions
  • No promise of every power-user workflow

The goal is simpler than that: give you the basics you need to get a script or file edit done quickly.

Core Features

Fast File Editing

Open a file, change it, save it, and continue without extra ceremony.

Simple Multi-Window Workflow

Keep different folders and files open in separate windows without mixing state.

Comfortable Defaults

Adjust fonts, theme, wrap mode, indentation, and other essentials.

Script-Friendly

Useful for shell, dotenv, Python, and PowerShell editing without extra setup.

Code is for when you want a capable editor that stays out of the way.
